# Vastraa 👗

Vastraa is a responsive fashion e-commerce website built using HTML, CSS, Bootstrap, and JavaScript.

## Features
- Responsive design
- Home page
- Product listing
- Product details
- User Login & Registration
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Project Structure
- index.html
- products.html
- product.html
- login.html
- register.html

## How to Run
1. Download or clone the repository.
2. Open `index.html` in your browser.
